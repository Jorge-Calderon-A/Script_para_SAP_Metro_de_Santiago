# Script de Python  para optimizar tiempo de ordenes y avisos en SAP para Metro de Santiago

En mi puesto de trabajo algunas veces urge optimizar tiempo para dedicar la aplicación y el foco del trabajador a otras actividades que ameriten mas enfoque para analizar ciertas cituaciones, ante esta problematica, tuve la necesidad de buscar una solución que tenga la capacidad de satisfacer ciertas acciones, mi opción principal fue crear un script para optimizar el tiempo en el que realizo un aviso y una orden en SAP, la versión que se presenta a continuación esta completamente desligada a SAP, no hay fuentes que se puedan extraer para intentar comprender la empresa, cabe destacar que este script esta bajo licencia en Github. El script esta realizado bajo el ambiente de Visual Studio Code, tambien puede ser aplicado desde PowerShell y simbolo de sistema.
