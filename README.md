# Script de Python  para optimizar tiempo y crear una estandarización de procesos en las creaciones de ordenes y avisos en SAP para Metro de Santiago

En optimización podemos comprender que uno de los ejes fundamentales dentro de nuestras acciones son llevadas bajo la optimización de procesos, estos procesos no solamente se pueden llevar a las empresas, tambien a nuestra vida, creando una vida que sea mas optima generando una eficiencia o eficacia para cualquier proyecto que se realice.

He tenido la necesidad de estandarizar un proceso de manera autonoma para crear ordenes y avisos en SAP dentro de mi area de trabajo, esta aplicación en su prototipo me dado la oportunidad de estructurar claramente todos los requerimientos para un anilsis mas completo de los mantenimientos, creando una enseñanza para percibir el entorno y observarlo mejor con todas las acciones y objetos que me rodean, entre ellos el tiempo, acciones humanas, objetos en el espacio presentado, clima, factores externos, entre otros.

Por ello he creado un script en Python para optimizar el tiempo y desarrollar mas las oportunidades de agregar ciertos factores que me ayuden a completar un trabajo como debe ser. El script esta desarrollado en Visual Studio Code, en Python y se encuentra como archivo de codigo abierto para el publico a parte de este readme.

A continuación se presenta un video donde explico la estructura del prototipo del script: https://youtu.be/0GT6-NxFWiw

Como se menciona anteriormente, el script no contiene información confidencial, debilidades para interpretar datos de la empresa, personas, objetos y lugares.

A la fecha el script se encuentra finalizado, se puede encontrar en el documento .py adjuntado (Programación Python.py).
